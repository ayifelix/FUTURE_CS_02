# Phishing Detection System - Task 2
**Stagiaire :** MAWOUÉNA LOUMON FÉLIX AYI  
**ID :** FIT/JAN26/CS5914

## Description
Ce projet consiste en une analyse complète d'un e-mail pour détecter des tentatives de phishing. 
Le rapport inclut :
- Une analyse visuelle des indices de fraude.
- Une analyse technique des en-têtes (Email Headers).
- La vérification des protocoles SPF, DKIM et DMARC.
